Pengendalian Multi Relay Lampu, Kipas, dan Charger Berbasis ESP32 Menggunakan Blynk

Deskripsi
Proyek ini merupakan tugas UAS Mata Kuliah Sistem Mikrokontroler. Sistem menggunakan ESP32 yang terhubung ke platform Blynk untuk mengendalikan tiga perangkat, 
yaitu lampu, kipas, dan charger melalui jaringan WiFi.

Fitur
- Mengontrol Lampu
- Mengontrol Kipas
- Mengontrol Charger
- Master Switch (menyalakan semua perangkat sekaligus)
- Sinkronisasi status perangkat dengan dashboard Blynk

Tools
- ESP32
- Wokwi Simulator
- Blynk IoT
- Arduino IDE

File Proyek
- `sketch.ino`
- `diagram.json`
- `libraries.txt`
